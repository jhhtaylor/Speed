Welcome to the README of the Speed Game (Prototype 1, Group 9, Game Design 3B, University of the Witwatersrand, 29 July 2019)

To enter the game, double click the exe for the Speed Game.

Jump using space and hop on the door latch the start the game scene. Avoid obstacles until you reach a high score, collect the pickup to enter Fist Mode.
